# AGENTS.md

## Project Overview

RAG Chatbot is a Retrieval-Augmented Generation application that allows users to query PDF documents using Large Language Models.

## Tech Stack

- Python
- Streamlit
- LangChain
- FAISS
- Gemini API

## Project Structure

- app.py : Main application
- utils/ : Helper functions
- tests/ : Unit tests
- docs/ : Documentation

## Development Guidelines

- Keep functions small and modular
- Add type hints
- Maintain test coverage
- Avoid hardcoded secrets
